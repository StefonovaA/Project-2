# <center>Project 2. Анализ резюме на HeadHunter</center> 

![](https://raw.githubusercontent.com/AndreyRysistov/DatasetsForPandas/main/hh%20label.jpg)

Проект: Анализ резюме из HeadHunter Данный проект посвящен анализу данных о резюме, полученных с сайта HeadHunter c использованием навыков SQL и Python


## Описание проекта

Наш проект включает в себя несколько этапов:

- знакомство с данными;
- предварительный анализ данных;
- детальный анализ вакансий;
- анализ работодателей;
- предметный анализ.

Каждая из частей состоит из блока практических заданий, которые необходимо выполнить в Jupyter-ноутбуке.


## Используемые инструменты

- **SQL**: Для выполнения запросов к базе данных PostgreSQL.
- **Python**: Для обработки данных и выполнения дополнительных вычислений.
- **Pandas**: Для анализа и манипуляции данными.
- **PostgreSQL**: База данных, содержащая информацию о вакансиях, работодателях и регионах.

## Анализ данных

### Предварительный анализ

В ходе предварительного анализа были получены следующие результаты:

- Всего в базе данных содержится 49197 вакансий.
- Количество работодателей достигает 23501.
- В данных информация по 1362 регионам.
- В базе представлены 294 сферы деятельности.

### Детальный анализ вакансий

1. Топ-5 регионов по количеству вакансий:

- Москва
- Санкт-Петербург
- Минск
- Новосибирск
- Алматы

2. Зарплатные вилки:

- Среднее значение нижней границы: 71065.
- Среднее значение верхней границы: 110537.

3. Формат занятости:

Наиболее популярный формат: Полная занятость с полным рабочим днем

4. Требуемый опыт работы:

Самый популярный уровень опыта: От 1 года до 3 лет.

### Анализ работодателей

**Выводы по анализу работодателей**

1. Яндекс находится на первом месте по количеству вакансий (1933 вакансии)

2. В Москве, Санкт-Петербурге и Минске наибольшее количество работодателей и вакансий. Регион Россия реже всего указывают в вакансиях.

3. Яндекс публикует вакансии в большем количестве регионов 

4. У 8419 работодателей не указана сфера деятельности

5. Компания 2ГИС находится на 3м месте среди компаний у которых указаны 4 сферы деятельности

6. 3553 компаний указали "Разработка программного обеспечения" в сфере своей деятельности

7. Компания Яндекс предствалена в городах миллионниках в общем количестве 485 вакансий 

### Предметный анализ
**Выводы по предметному анализу**

1. Общее число вакансий, имеющих отношение к данным, состовляет 1771, что говорит высоком спросе на специалистов в сфере анализа данных.

2. Для начинающих (junior) дата-сайентистов доступно 51 подходящая вакансия.

3. Вакансий, где "SQL" или "postgre" указаны как ключевые навыки, составляет 229, что свидетельствует о важности владения этими инструментами.

4. Вакансий, где "Python" указан как ключевой навык, составляет 351, что свидетельствует о его важности как одного из основных инструментов в сфере анализа данных.

5. Среднее количество ключевых навыков, предъявляемых к специалисту, превышает 6, что говорит о необходимости обладания разносторонними знаниями и умениями.

6. Специалистам со стажем от 3 до 6 лет предлагается средняя зарплату в размере 243115, что делает професию финансово привлекательной

### Общие выводы по проекту

1.  Всего в базе данных содержится 49197 вакансий. Количество работодателей достигает 23501. Указана информация по 1362 регионам в рамках 294 направлений работы
2. Среднее значение нижней границы зарплатной вилки: 71065, верхней границы: 110537. Наиболее популярный формат занятости: Полная занятость с полным рабочим днем. Самый популярный уровень опыта работы: От 1 года до 3 лет.
3. Компания "Яндекс" возглавляет список по количеству вакансий, предлагая 1933 рабочих места. Отдельного внимания заслуживает сфера работы с данными, где насчитывается 1771 вакансия. По данным анализа следует, что навуки Python и SQL являются одними из основных навыков для специалиста. В среднем дата-сайентист должен обладать более чем шестью ключевыми компетенциями.
4. Наибольшее количество вакансий приходится на область разработки программного обеспечения (в т.ч и в рамках удаленной работы)
